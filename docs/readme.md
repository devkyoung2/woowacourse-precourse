## 구현할 기능 목록

- [x] 게임을 시작한다.
- [] 컴퓨터는 랜덤 수를 생성한다.
  - [] 서로 다른 임의의 수 3개 생성
- [] 사용자에게 숫자를 입력받는다.
- [] 숫자를 비교한다.
  - [] 같은 수가 같은 자리에 있으면 스트라이크
  - [] 다른 자리에 있으면 볼
  - [] 같은 수가 전혀 없으면 낫싱
- [] 3개의 숫자를 모두 맞추면 게임을 종료한다.
- [] 게임 재시작 여부를 입력받는다.
  - [] 1가 입력되면 게임을 재시작한다.
  - [] 2이 입력되면 게임을 종료한다.
- [] 에러를 처리한다.
