## 구현할 기능 목록

- [x] 게임을 시작한다.
- [x] 컴퓨터는 랜덤 수를 생성한다.
  - [x] 서로 다른 임의의 수 3개 생성
- [x] 사용자에게 숫자를 입력받는다.
  - [x] 사용자는 서로 다른 3개의 수를 입력한다.
- [x] 숫자를 비교한다.
  - [x] 같은 수가 같은 자리에 있으면 스트라이크
  - [x] 다른 자리에 있으면 볼
  - [x] 같은 수가 전혀 없으면 낫싱
- [x] 3개의 숫자를 모두 맞추면 게임을 종료한다.
- [x] 게임 재시작 여부를 입력받는다.
  - [x] 1가 입력되면 게임을 재시작한다.
  - [x] 2이 입력되면 게임을 종료한다.
- [x] 에러를 처리한다.

---

- 컨트롤러
  - 게임을 시작한다.
  - 게임을 종료한다.
  - 게임을 재시작한다.
- 사용자
  - 숫자를 추측한다.
- 컴퓨터
  - 랜덤수를 생성한다.
  - 숫자를 비교한다.(공에게 요청)
- 공
  - 세자리 랜덤수
